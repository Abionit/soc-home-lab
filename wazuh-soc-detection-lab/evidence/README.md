# Evidence Guide

Recommended evidence for portfolio review:

- Wazuh dashboard login screen after deployment
- Wazuh manager/indexer/dashboard containers running
- Custom rules copied into the Wazuh manager
- Generated lab events
- Triggered alerts in Wazuh
- MITRE ATT&CK fields in alert details
- Alert analytics outputs generated by Python
- Optional Streamlit dashboard showing SOC metrics

Keep screenshots focused on evidence, not private credentials or local secrets.
